# Метрохим — прототип и КП

Презентационные материалы для компании «Метрохим» (аренда/химчистка/пошив спецодежды, СПб).

- `kp.html` — коммерческое предложение (стратегия онлайн-развития)
- `index.html` — кликабельный прототип сайта

Кнопка «Посмотреть прототип сайта» в КП ведёт на прототип (`index.html`).

Опубликовано через GitHub Pages.
